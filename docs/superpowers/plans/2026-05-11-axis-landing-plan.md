# Axis Landing Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Переработать текущий одностраничный лендинг в `index.html` под бренд **Axis** в светлом стиле (вариант B Soft Minimal), используя референс-изображение как фоновый мотив, и привести структуру страницы к утверждённым блокам (оффер → боль/решение → услуги → преимущества → процесс → формы (2) → отзывы/кейсы).

**Architecture:** Без сборщика: один HTML-файл со встроенными CSS/JS. Визуальный стиль задаётся через CSS-переменные, блоки — семантическими секциями. JS — только для меню, плавного скролла, reveal-анимации, минимальной валидации формы.

**Tech Stack:** HTML5 + CSS (custom properties) + Vanilla JS (ES6). Подключение шрифтов через Google Fonts.

---

## File Structure (что трогаем)
- Modify: `index.html` (замена контента/стилей/JS под Axis + светлый стиль)
- Create: `assets/axis-reference.png` (копия референса из чата; используется в hero как background-image)
- Modify (optional): `.gitignore` (добавить `.superpowers/`, чтобы не коммитить артефакты визуального компаньона)

## UX/Design decisions locked-in
- Тема: светлая (Soft Minimal), один основной акцент (индиго/синий) + графитовый текст
- Hero: H1 + подзаголовок + CTA + **мини-форма на первом экране**
- Второй CTA: **полная форма** внизу (дублирование)
- Референс-фото: очень лёгкий фон (opacity + blur через псевдо-слой/фильтры)
